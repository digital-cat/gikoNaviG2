------------------------------
はじめに
------------------------------
このソフトウェアは、巨大掲示板２ちゃんねる用ブラウザです。
ギコナビプロジェクトが活動停止中のため、当面のギコナビの維持管理を目的としている
のがこのギコナビ(避難所版)です。
本家ギコナビプロジェクトの再開を祈りつつ、メンテナンスしていきます。
本家ギコナビ最終版（ﾊﾞﾀ63）のreadme.txtも同梱していますので、こちらも一読ください。
最新版は、以下のWebサイトでチェックしてください。

ギコナビサイト
http://gikonavi.sourceforge.jp/

ギコナビ(避難所版)サイト
http://gikonavigoeson.sourceforge.jp/


------------------------------
転載について
------------------------------
Webや雑誌などへの転載は自由に行ってください。
作者への連絡は不要です。


------------------------------
動作環境
------------------------------
OS: Windows2000以上 (2000, XP, 2003, Vista(*1), 7(*1), 8(*1), 8.1(*1)）
*1 "Program Files"、"Program Files (x86)"フォルダ以外にインストールしてください。
CPU: 速ければ速いほど速く動くような気がする
ﾒﾓﾘ: 64MB以上（多ければ多いほどいいような気がする）
ｿﾉﾀ: 800x600以上の解像度、ﾊｲｶﾗｰ以上、ポインティングデバイス必須
     InternetExplorer5.5以上必須（ﾊﾞﾀ64よりバージョンを引き上げています）

Windows98, 98SE, Meは、ﾊﾞﾀ66より動作しなくなりました。
WindowsVistaより古いOSでは動作確認を行いません。
また、サポートの切れたOSでインターネットに接続する行為は危険なためお勧めしません。
